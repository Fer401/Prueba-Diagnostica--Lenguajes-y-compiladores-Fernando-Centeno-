## Descripción del Repositorio
Este repositorio contiene la resolución algorítmica de los tres planteamientos exigidos en la prueba diagnóstica, desarrollados íntegramente en Python 3.

## Estructura del Proyecto
Cumpliendo con los lineamientos de entrega, el proyecto se divide en 4 carpetas principales:
* `problema1/`: Contiene el código fuente del Analizador Léxico de expresiones aritméticas.
* `problema2/`: Contiene el validador de cadenas bajo la Notación FEN (Forsyth-Edwards Notation).
* `problema3/`: Contiene el script demostrativo de la Conjetura de Collatz evaluada en un intervalo [p, q].
* `recursos/`: Contiene información adicional de la prueba diagnóstica.

## Instalación y Dependencias
El código fuente fue desarrollado utilizando exclusivamente la biblioteca estándar de Python (específicamente el módulo `re`). 
No es necesaria la instalación de paquetes externos mediante gestores como `pip`. 

**Para ejecutar las soluciones:**
1. Asegúrese de tener Python 3.x instalado en su sistema.
2. Navegue desde la terminal a la carpeta del problema deseado.
3. Ejecute el script (Ejemplo. `python problema1.py`).
